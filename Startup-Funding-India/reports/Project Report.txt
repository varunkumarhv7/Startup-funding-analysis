# 2. Objective

The objective of this project is to analyze startup funding trends in India and build an interactive dashboard that enables users to explore funding patterns based on various filters. The dashboard helps identify investment trends across cities, industries, years, startups, and investment types.

---

# 3. Problem Statement

India has witnessed significant growth in startup investments over the past decade. Analyzing this large amount of funding data manually is difficult. This project provides an interactive visualization platform to understand:

* Which cities receive the highest funding
* Which industries attract the most investment
* Funding trends over the years
* Popular investment types
* Leading startups and investors

# 5. Dataset

The dataset contains startup funding information with the following columns:

* Startup
* Industry
* Sub Vertical
* City
* Investors
* Investment Type
* Investment Amount (USD)
* Date

---

# 6. Data Preprocessing

The following preprocessing steps were performed:

* Removed missing values
* Converted Date column into datetime format
* Created a new Year column
* Cleaned Investment Amount values
* Removed duplicate records
* Standardized column names
* Converted data into analysis-ready format
